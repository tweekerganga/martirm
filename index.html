What is Project LawOS?

Project LawOS is an open source C++ operating system made by Martin Szokai in January 2026.

Updates are posted in the LawOS Discord. Current version is 33.2 LawBuild - the first desktop version with BIOS support.

Warning: The OS for now is very unstable and may not boot. All versions only work with BIOS, NOT UEFI LawOS is an experimental OS to make Martin 's coding skills better.
